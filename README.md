# Mkdocs 

## Run the following once 

Installation and initial setup of mkdocs 

### Installation

```
python3 -m venv ~/.virtualenvs/mkdocs

. ~/virtualenvs/mkdocs/bin/activate

pip install --upgrade pip

pip install mkdocs
```
